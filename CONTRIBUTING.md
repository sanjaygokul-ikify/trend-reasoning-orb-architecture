# Contributor Guide
1. Fork and clone the repository
2. Create feature branch
3. Run `make test` locally
4. Submit PR with changelog entry

## Development Workflow
1. Feature proposal via RFC PR
2. Design review with maintainers
3. Implementation in atomic commits
4. Integration testing with `make ci`

## Communication
- Discuss on Discord #dev channel
- Code reviews follow Google-style guide
- Breaking changes require RFC approval
